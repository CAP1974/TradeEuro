# TradeEuro V9

Estrutura modular para GitHub Pages.

## Ficheiros
- `index.html` → estrutura da app
- `style.css` → visual
- `app.js` → lógica
- `data.json` → dados da carteira

## Atualização diária
Na maioria dos dias, basta atualizar `data.json`.

## Publicação no GitHub Pages
1. Colocar estes ficheiros na raiz do repositório
2. Settings > Pages
3. Source: Deploy from a branch
4. Branch: `main`
5. Folder: `/ (root)`
6. Save
